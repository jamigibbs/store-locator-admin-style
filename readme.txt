=== Description ===

This plugin will apply custom style to the Store Locator Plus plugin.

=== Installation ===

1. Upload the plugin to `/wp-content/plugins/`. Alternately, you can download this package as a zip and upload it via the WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress. That's it!
